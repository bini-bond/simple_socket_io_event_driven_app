This app is simple chatroom socket io, event driven architecture, node express app.

it uses ejs rendering engine and the view is found in views folder by the name "chat.ejs"
the severside socket_io app is found in socket folder by the name "socket.js"
the server is found in ./bin/www .
dependencies  and modules are not included since they are large in size.
